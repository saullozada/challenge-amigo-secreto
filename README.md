# Challenge Amigo Secreto

## Descripción
Este proyecto es una aplicación desarrollada en JavaScript, donde el usuario ingresa el nombre de sus amigos y está devolvera el nombre del amigo secreto.

## Funcionalidades
- Permite añadir un numero indefinido de registros.
- El registro minimo de nombres son: 2
- No permite añadir registros vacios " ".
- Valida que no contengan caracteres especiales.
- Muestra el listado de amigos.
- Sortea el amigo secreto de forma aleatoria de acuerdo a la funcion.

## Instrucciones para clonar el proyecto
